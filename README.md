# Cafe Billing System

## Description

Cafe Billing System is a Python command-line application that simulates a simple café ordering and billing process. Customers can choose items from a predefined menu, and the program calculates the total bill by applying an 8% GST to each purchased item.

This project is intended for beginners who want to practice Python dictionaries, loops, conditional statements, arithmetic operations, and user interaction through a command-line interface.

---

## Features

* Display a café menu with item prices
* Accept customer orders
* Calculate item prices automatically
* Apply an 8% GST to each purchased item
* Generate the total bill
* Handle unavailable menu items gracefully
* Simple command-line interface

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Cafe-Billing-System.git
```

2. Navigate to the project folder.

```bash
cd Cafe-Billing-System
```

3. Run the program.

```bash
python cafe_billing.py
```
